# ☕ Coffee Shop Sales Analysis Dashboard

## 📌 Project Overview

This project focuses on analyzing coffee shop sales data using **MySQL** and visualizing insights with **Power BI**. The goal is to identify business trends, customer behavior patterns, and key performance metrics to support better decision-making.

The project includes data cleaning, SQL analysis, KPI calculations, and an interactive Power BI dashboard.

---

## 🛠 Technologies Used

- MySQL
- Power BI
- DAX
- Power Query
- Data Modeling

---

## 📊 Dashboard Features

- Total Sales KPI
- Total Orders KPI
- Total Quantity Sold KPI
- Month-over-Month (MoM) Growth Analysis
- Daily Sales Trends
- Sales by Product Category
- Top 10 Products by Sales
- Sales by Store Location
- Weekday vs Weekend Analysis
- Sales by Days & Hours Heatmap
- Interactive Filters
